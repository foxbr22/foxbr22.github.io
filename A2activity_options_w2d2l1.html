Listening Comprehension: A Day in Tech Support

1. What is the speaker's job?

2. What does the agent do every morning?

3. What was the problem with the customer's speakers?

4. What suggestion did the agent make to the user whose video kept breaking up?

5. What kind of attempt did the agent deal with before lunch?

6. What did the agent host at the end of the day?

7. What service was the customer trying to sign up for?

8. What did the agent help the client do after the hacking attempt?

9. How does the agent describe the tickets waiting in the morning?

10. What does the agent say at the end of the monologue?
